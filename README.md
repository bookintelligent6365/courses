# BookIntelligent Course Site

โฟลเดอร์นี้แยกเป็น 2 ส่วนหลัก:

```text
.
├── docs/                 เอกสารแผนและสถานะโปรเจกต์
├── site/                 เว็บคอร์สสำหรับนำไปวางบน GitHub Pages
│   ├── index.html        หน้ารวมคอร์สทั้งหมด
│   ├── courses/          หน้ารายละเอียดแต่ละคอร์ส
│   └── assets/
│       └── css/styles.css
└── .claude/              config เดิมของ local assistant
```

## เปิดดูเว็บ

เปิดไฟล์นี้ใน browser:

```text
site/index.html
```

## Deploy ขึ้น GitHub Pages

ให้อัปโหลดทั้งโฟลเดอร์ `site/` ขึ้น repo หรือย้ายไฟล์ใน `site/` ไปไว้ที่ root ของ repo สำหรับ GitHub Pages

## เอกสารโปรเจกต์

- `docs/Plan.md`
- `docs/Progress Status.md`
- `docs/Asset.txt`
