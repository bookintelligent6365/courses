รายละเอียดคอร์ส

คอร์สนี้เหมาะกับคนที่ใช้ Excel ทำรายงานอยู่แล้ว แต่อยากเปลี่ยนไฟล์ตารางธรรมดาให้เป็น Dashboard ที่กด Filter ได้ อ่านง่าย และอัปเดตได้ทุกเดือนโดยไม่ต้องสร้างใหม่ตั้งแต่ต้น

เนื้อหาครอบคลุมตั้งแต่การเตรียมข้อมูลด้วย Excel Table, ใช้ PivotTable และ PivotChart สร้างกราฟ, ติด Slicer และ Timeline สำหรับ interactive report และออกแบบ layout Dashboard ให้ดูเป็นมืออาชีพ

จบคอร์สนี้คุณจะมีไฟล์ Dashboard จริง 3 แบบ พร้อม template ที่นำไปปรับใช้กับข้อมูลของตัวเองได้ทันที

ผลลัพธ์หลังเรียนจบ

Sales Dashboard

ดูยอดขายรายเดือน สินค้าขายดี ยอดขายตาม region และ trend เทียบเป้าหมาย

HR Attendance Dashboard

สรุปการเข้าทำงาน วันลา มาสาย และรายชื่อพนักงานรายเดือน

Inventory Dashboard

ติดตามสต็อกสินค้า สินค้าใกล้หมด มูลค่าสต็อก และการเคลื่อนไหวสินค้า

ประโยชน์ที่ผู้เรียนจะได้รับ

ทำรายงาน Excel ให้กด Filter และอัปเดตได้เอง ใช้ PivotTable / PivotChart สร้างกราฟสรุปข้อมูลได้คล่อง ติด Slicer และ Timeline สำหรับ interactive report ใช้สูตร XLOOKUP, SUMIFS, COUNTIFS ในงานจริง ออกแบบ Dashboard layout ที่ใช้ประชุมได้จริง ทำไฟล์ให้อัปเดตง่ายและไม่พังเมื่อเพิ่มข้อมูล

คอร์สนี้เหมาะกับใคร

  • พนักงานออฟฟิศที่ต้องทำรายงานยอดขาย รายจ่าย หรือรายงานประจำเดือน
  • Sales Admin, HR และ Accounting ที่ต้องสรุปข้อมูลให้ผู้บริหารอ่าน
  • Analyst มือใหม่ที่ต้องการยกระดับการนำเสนอรายงาน
  • คนที่ใช้ Excel อยู่แล้วและอยากเริ่มทำ Dashboard โดยไม่เขียนโค้ด

ผู้เรียนควรมีพื้นฐานอะไรก่อน

  • ใช้ Excel พื้นฐานได้ เช่น กรอกข้อมูล จัด format และใช้สูตรง่าย ๆ
  • เคยใช้ SUM, AVERAGE หรือ IF มาบ้างจะช่วยให้เรียนได้เร็วขึ้น
  • ไม่จำเป็นต้องเคยใช้ PivotTable หรือเขียนโค้ด VBA มาก่อน

เนื้อหาคอร์ส

Module 1: เตรียมข้อมูลด้วย Excel Table 4 บท
  1. ภาพรวมคอร์สและ Dashboard ที่จะสร้าง
  2. ทำไม Excel Table ถึงสำคัญ และวิธีสร้าง
  3. จัดการ header, data type และ format ให้ถูกต้อง
  4. เทคนิคทำความสะอาดข้อมูลก่อนสร้าง Pivot
Module 2: สูตรสำคัญสำหรับรายงาน 5 บท
  1. SUMIFS และ COUNTIFS สำหรับสรุปตามเงื่อนไข
  2. XLOOKUP แทน VLOOKUP ใช้งานง่ายกว่า
  3. IF / IFS สำหรับจัดหมวดหมู่และสถานะ
  4. TEXT, MONTH, YEAR สำหรับจัดการวันที่
  5. IFERROR และการป้องกันสูตรแสดง error
Module 3: PivotTable และ PivotChart 5 บท
  1. สร้าง PivotTable ตั้งแต่ต้นและตั้งค่าสำคัญ
  2. จัด Row, Column, Values และ Filter field
  3. Grouping วันที่ เดือน ไตรมาส ปี
  4. สร้าง PivotChart และเลือก chart type ที่เหมาะกับข้อมูล
  5. ปรับแต่ง PivotChart ให้ดูเป็นมืออาชีพ
Module 4: Slicer, Timeline และ Interactive Report 4 บท
  1. เพิ่ม Slicer สำหรับ filter ตามหมวดหมู่
  2. เพิ่ม Timeline สำหรับ filter ตามช่วงเวลา
  3. เชื่อม Slicer กับ Pivot หลายตัวพร้อมกัน
  4. ออกแบบ Slicer และ Timeline ให้ดูสวยงาม
Module 5: Dashboard Layout และ Design 5 บท
  1. วาง layout Dashboard บน worksheet
  2. สร้าง KPI Card ด้วยสูตรและ formatting
  3. Conditional Formatting สำหรับ highlight ข้อมูลสำคัญ
  4. ปรับสี ฟอนต์ และระยะห่างให้อ่านง่าย
  5. ซ่อน gridline, row/column header และปรับ zoom ให้เหมาะกับการนำเสนอ
Module 6: เคสจริง 3 แบบ 3 บท
  1. Sales Dashboard: ยอดขายรายเดือน สินค้าขายดี และ trend
  2. HR Attendance Dashboard: การเข้างาน วันลา และมาสาย
  3. Inventory Dashboard: สต็อกสินค้า สินค้าใกล้หมด และมูลค่าสต็อก
Module 7: ทำไฟล์ให้อัปเดตง่ายและใช้ต่อได้จริง 3 บท
  1. วิธีเพิ่มข้อมูลใหม่โดยไม่ให้ Pivot และ Chart เสีย
  2. ป้องกันสูตรและ format ไม่ให้ถูกแก้โดยไม่ตั้งใจ
  3. Checklist Dashboard ก่อนส่งหรือนำเสนอ
