รายละเอียดคอร์ส

คอร์สนี้ออกแบบสำหรับคนที่ใช้ Google Sheets ทำงานอยู่แล้ว แต่อยากยกระดับจากตารางธรรมดาให้เป็น Dashboard ที่สรุป KPI, กราฟ และตัวกรองข้อมูลได้ในหน้าเดียว

เนื้อหาจะเน้น project end-to-end ไม่ใช่สอนสูตรแยกเป็นชิ้น ๆ โดยจะเริ่มจาก raw data จัดโครงสร้างข้อมูล สรุปด้วยสูตรสำคัญ แล้วประกอบเป็น Dashboard 3 เคส ได้แก่ Sales, Expense และ Task

เป้าหมายคือผู้เรียนต้องทำไฟล์ Dashboard ของตัวเองได้หลังเรียนจบ และเข้าใจวิธีปรับ template ไปใช้กับข้อมูลของตัวเอง

ผลลัพธ์หลังเรียนจบ

ดูยอดขายรวม ยอดขายรายเดือน สินค้าขายดี และยอดขายตามหมวดหมู่

สรุปรายรับรายจ่าย กำไร/ขาดทุน และค่าใช้จ่ายตามประเภท

Task Dashboard

ติดตามงานค้าง งานเสร็จ งานใกล้ครบกำหนด และสถานะของทีม

ประโยชน์ที่ผู้เรียนจะได้รับ

ลดเวลาทำรายงานซ้ำ ๆ ทุกสัปดาห์หรือทุกเดือน เปลี่ยนข้อมูลดิบให้เป็น Dashboard ที่อ่านง่าย ใช้สูตรสำคัญกับงานจริง ไม่ใช่จำสูตรแยกเป็นชิ้น สร้างรายงาน Sales, Expense และ Task ได้เอง ได้ไฟล์ template พร้อมนำไปปรับใช้ต่อ เข้าใจวิธีแชร์ไฟล์และป้องกันสูตรพัง

คอร์สนี้เหมาะกับใคร

  • พนักงานออฟฟิศที่ต้องทำรายงานยอดขาย รายจ่าย หรืองานประจำ
  • เจ้าของธุรกิจ SME ที่อยากเห็นภาพรวมธุรกิจจาก Google Sheets
  • ฟรีแลนซ์หรือแอดมินที่ต้องทำไฟล์รายงานให้ทีม/ลูกค้า
  • คนที่เคยดูคลิปฟรีแล้วอยากเรียนแบบเป็น project ตั้งแต่ต้นจนจบ

ผู้เรียนควรมีพื้นฐานอะไรก่อน

  • ใช้ Google Sheets พื้นฐานได้ เช่น กรอกข้อมูล จัด format และใช้สูตรง่าย ๆ
  • เคยใช้สูตรเช่น SUM หรือ IF มาบ้างจะเรียนได้เร็วขึ้น
  • ไม่จำเป็นต้องเขียนโค้ด และไม่จำเป็นต้องเคยทำ Dashboard มาก่อน

เนื้อหาคอร์ส

Module 1: เตรียมพื้นฐานและเป้าหมาย Dashboard 4 บท
  1. ภาพรวมคอร์สและ Dashboard ที่จะสร้าง
  2. Dashboard ที่ดีควรตอบคำถามอะไร
  3. โครงสร้างไฟล์: Raw Data, Summary, Dashboard
  4. วิธีใช้ Practice File และ Final Template
Module 2: เตรียมข้อมูลให้พร้อมวิเคราะห์ 5 บท
  1. รูปแบบข้อมูลที่ดี: 1 row = 1 transaction
  2. ทำความสะอาดข้อมูลเบื้องต้น
  3. จัดการวันที่ เดือน ปี และหมวดหมู่
  4. ทำ Data Validation และ Dropdown
  5. จัดระเบียบ Sheet และป้องกันสูตรพัง
Module 3: สูตรสำคัญสำหรับ Dashboard 5 บท
  1. SUMIFS / COUNTIFS สำหรับสรุป KPI
  2. FILTER สำหรับดึงรายการตามเงื่อนไข
  3. QUERY สำหรับรายงานแบบยืดหยุ่น
  4. Dynamic Filter ด้วย cell input
  5. IF / IFS สำหรับสถานะและเงื่อนไข
Module 4: สร้าง Dashboard ตัวแรก 7 บท
  1. วาง Layout หน้า Dashboard
  2. สร้าง KPI Cards: ยอดขาย รายจ่าย กำไร งานค้าง
  3. สร้างกราฟแนวโน้มรายเดือน
  4. สร้างกราฟตามหมวดหมู่
  5. ทำ Filter เลือกเดือน / หมวดหมู่ / สถานะ
  6. Conditional Formatting — highlight ค่าเกิน/ต่ำกว่าเป้า
  7. ปรับสี ฟอนต์ และระยะห่างให้อ่านง่าย
Module 5: เคสจริง 3 แบบ 6 บท
  1. Sales Dashboard (1): เตรียม data + สูตร KPI ยอดขาย
  2. Sales Dashboard (2): build layout, chart และส่งมอบ
  3. Personal Finance Dashboard (1): เตรียม data รายรับรายจ่ายส่วนตัว
  4. Personal Finance Dashboard (2): build Dashboard วางแผนการเงินส่วนตัว
  5. Task Dashboard (1): เตรียม data + สูตรติดตามสถานะงาน
  6. Task Dashboard (2): build layout และ filter ตามทีม/วันครบกำหนด
Module 6: ส่งมอบและใช้งานจริง 3 บท
  1. วิธีแชร์ไฟล์และตั้งสิทธิ์ให้ปลอดภัย
  2. วิธีทำสำเนา Template ไปใช้กับงานของตัวเอง
  3. Checklist ตรวจ Dashboard ก่อนใช้งานจริง
