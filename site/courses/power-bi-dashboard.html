รายละเอียดคอร์ส

คอร์สนี้เหมาะกับคนที่ต้องทำรายงานธุรกิจหรือต้องการยกระดับจาก Excel ไปสู่ Business Intelligence ด้วย Power BI โดยจะพาเรียนแบบ project-based ตั้งแต่ต้นจนจบ ไม่ใช่แค่สาธิต feature

เนื้อหาครอบคลุม Power Query สำหรับ cleaning data, การสร้าง Data Model และ Relationship, เขียน DAX Measure ที่ใช้บ่อยในงานจริง และออกแบบ Dashboard ที่สวยงามสำหรับผู้บริหาร

จบคอร์สนี้คุณจะมีไฟล์ PBIX Dashboard 3 แบบพร้อม template และ DAX Cheat Sheet ที่ใช้ต่อได้ทันที

ผลลัพธ์หลังเรียนจบ

Sales Dashboard

ดูยอดขายรายเดือน region สินค้าขายดี และ trend เทียบ target พร้อม Slicer กรองข้อมูล

Market Share Dashboard

วิเคราะห์ส่วนแบ่งตลาด เปรียบเทียบ performance ระหว่าง product และ segment

Executive Dashboard

สรุป KPI หลักสำหรับผู้บริหาร ออกแบบให้อ่านได้ใน 30 วินาที

ประโยชน์ที่ผู้เรียนจะได้รับ

เตรียมข้อมูลจาก Excel/CSV ด้วย Power Query สร้าง Data Model และ Relationship ได้ถูกต้อง เขียน DAX Measure สำหรับ KPI จริง สร้าง KPI Card, Chart, Matrix, Slicer ออกแบบ Dashboard สำหรับผู้บริหาร Publish, Export และ Refresh Report

คอร์สนี้เหมาะกับใคร

  • Analyst, Data Analyst มือใหม่ที่ต้องการเรียนรู้ Power BI
  • Manager หรือ ผู้บริหารที่ต้องสร้างรายงานธุรกิจให้ทีม
  • เจ้าของธุรกิจที่ต้องการ Dashboard ดูภาพรวมกิจการ
  • คนที่ใช้ Excel ทำรายงานอยู่และอยากยกระดับสู่ BI tool

ผู้เรียนควรมีพื้นฐานอะไรก่อน

  • ใช้ Excel ได้ในระดับพื้นฐาน เช่น เข้าใจ table, row/column และการใช้ filter
  • เข้าใจ concept ของ PivotTable จะช่วยให้เข้าใจ Power BI ได้เร็วขึ้น
  • ไม่จำเป็นต้องเคยใช้ Power BI หรือเขียน DAX มาก่อน

เนื้อหาคอร์ส

Module 1: เตรียมข้อมูลจาก Excel/CSV 4 บท
  1. ภาพรวมคอร์สและ Dashboard ที่จะสร้าง
  2. ติดตั้ง Power BI Desktop และตั้งค่าเบื้องต้น
  3. Connect ข้อมูลจาก Excel และ CSV
  4. ตรวจสอบและเข้าใจ data ก่อน load เข้า Power BI
Module 2: Power Query สำหรับ Cleaning Data 5 บท
  1. Power Query Editor: ภาพรวมและ interface
  2. เปลี่ยน data type, rename column และ remove error
  3. Filter, sort และ remove duplicate rows
  4. Split column, Merge column และ Custom Column
  5. Append และ Merge queries หลายตาราง
Module 3: Data Model และ Relationship 4 บท
  1. Star Schema: Fact table และ Dimension table คืออะไร
  2. สร้าง Relationship ระหว่างตาราง
  3. ตั้งค่า Relationship Direction และ Cardinality
  4. สร้าง Date Table สำหรับวิเคราะห์ตามเวลา
Module 4: DAX พื้นฐานและ Measure สำคัญ 6 บท
  1. DAX คืออะไร และต่างจาก Excel อย่างไร
  2. Measure vs Calculated Column
  3. SUM, COUNT, AVERAGE, DIVIDE
  4. CALCULATE และ Filter Context
  5. Time Intelligence: YTD, MTD, SAMEPERIODLASTYEAR
  6. IF, SWITCH, ISBLANK สำหรับ logic ใน Measure
Module 5: สร้าง Visual และ Dashboard 6 บท
  1. KPI Card, Multi-row Card
  2. Bar Chart, Line Chart, Combo Chart
  3. Matrix: แสดงข้อมูลแบบตาราง cross-tab
  4. Slicer: กรองข้อมูลแบบ interactive
  5. Drill-through และ Tooltip
  6. ออกแบบ layout หน้า Dashboard
Module 6: เคสจริง 3 Dashboard 3 บท
  1. Sales Dashboard: ยอดขาย, target, top products และ region
  2. Market Share Dashboard: ส่วนแบ่งตลาดและการเปรียบเทียบ
  3. Executive Dashboard: KPI summary สำหรับผู้บริหาร
Module 7: Dashboard UX สำหรับผู้บริหาร 4 บท
  1. หลักการออกแบบ Dashboard ที่อ่านง่ายใน 30 วินาที
  2. เลือก visual ที่เหมาะกับข้อมูลแต่ละประเภท
  3. ใช้สี, font และ spacing อย่างถูกหลัก
  4. Checklist ก่อนนำเสนอ Dashboard จริง
Module 8: Publish, Export และ Refresh 4 บท
  1. Publish ขึ้น Power BI Service
  2. Export เป็น PDF และ PowerPoint
  3. ตั้ง Data Refresh อัตโนมัติ
  4. แชร์ report และตั้งสิทธิ์ผู้ใช้
