รายละเอียดคอร์ส

คอร์สนี้พาคุณจาก Google Sheets ธรรมดา ไปเป็นระบบ Web App ที่มี Form รับข้อมูล, บันทึกลง Sheet, ค้นหา, แก้ไข และส่งอีเมลอัตโนมัติได้ โดยใช้ Google Apps Script เขียนโค้ดบน browser ไม่ต้องติดตั้งอะไรเพิ่ม

เนื้อหาออกแบบให้เริ่มจากพื้นฐาน JavaScript ที่จำเป็น แล้วค่อย ๆ ประกอบเป็นระบบจริงแบบ step-by-step จนถึงการ deploy เป็น Web App ที่คนอื่นใช้ได้จาก browser

ไม่จำเป็นต้องมีพื้นฐาน programming มาก่อน ขอแค่ใช้ Google Sheets ได้และพร้อมเรียนรู้การเขียนโค้ดเบื้องต้น

ระบบที่จะสร้างได้หลังเรียนจบ

Lead Form / ลงทะเบียน

Form รับข้อมูลลูกค้า บันทึกลง Sheet พร้อมส่ง email ยืนยันอัตโนมัติ

Booking System

ระบบจองคิว จองห้อง หรือจองบริการ พร้อมตรวจสอบความพร้อมก่อนยืนยัน

PO Request / ขออนุมัติ

ระบบส่งคำขออนุมัติพร้อม email แจ้งเตือนผู้อนุมัติอัตโนมัติ

ประโยชน์ที่ผู้เรียนจะได้รับ

ใช้ Google Sheets เป็น database ของระบบ เขียน Apps Script เบื้องต้นได้จริง สร้าง HTML Form รับข้อมูลและบันทึกลง Sheet ทำ Auto ID, Timestamp, Search และ Edit ส่งอีเมลอัตโนมัติเมื่อมีข้อมูลใหม่ Deploy Web App ให้คนอื่นใช้งานได้จาก browser

คอร์สนี้เหมาะกับใคร

  • เจ้าของธุรกิจ SME ที่ต้องการระบบรับข้อมูล form หรือ booking ใช้เอง
  • Admin หรือ Coordinator ที่ต้องการลดงาน manual ด้วย automation
  • ฟรีแลนซ์ที่อยากสร้างระบบให้ลูกค้าโดยไม่ต้องใช้ server หรือ backend
  • คนที่เคยทำ Google Sheets แล้วอยากต่อยอดสู่ระบบ automation แบบง่าย

ผู้เรียนควรมีพื้นฐานอะไรก่อน

  • ใช้ Google Sheets ได้ในระดับกลาง เช่น ใช้สูตร IF, VLOOKUP หรือ SUMIFS ได้บ้าง
  • เคยเห็น HTML หรือ JavaScript มาบ้างจะช่วยให้เรียนง่ายขึ้น แต่ไม่บังคับ
  • ไม่จำเป็นต้องมีพื้นฐาน programming หรือ server-side code มาก่อน

เนื้อหาคอร์ส

Module 1: Google Sheets เป็น Database 4 บท
  1. ภาพรวมคอร์สและ Web App ที่จะสร้าง
  2. โครงสร้าง Sheet ที่ดีสำหรับเป็น database
  3. การตั้งชื่อ column, data type และ header
  4. Named Range และ Dropdown สำหรับ validation
Module 2: พื้นฐาน Google Apps Script 5 บท
  1. Apps Script คืออะไร และเข้าถึงได้อย่างไร
  2. JavaScript พื้นฐานที่จำเป็น: variable, function, array, object
  3. อ่านและเขียนข้อมูลลง Google Sheet ด้วย Script
  4. วนซ้ำข้อมูลด้วย loop และเงื่อนไข if/else
  5. Debug และแก้ error เบื้องต้นใน Apps Script
Module 3: สร้าง HTML Form และรับข้อมูล 5 บท
  1. HTML Form เบื้องต้น: input, select, textarea, button
  2. ส่งข้อมูลจาก Form ไป Apps Script ด้วย google.script.run
  3. บันทึกข้อมูลลง Sheet และแจ้งผลสำเร็จ/ล้มเหลว
  4. Validation ข้อมูลก่อนบันทึก
  5. สร้าง Form ที่ดูดีด้วย CSS เบื้องต้น
Module 4: Auto ID, Timestamp, Search และ Edit 6 บท
  1. สร้าง Auto ID อัตโนมัติทุกครั้งที่บันทึก
  2. เพิ่ม Timestamp วันที่และเวลาอัตโนมัติ
  3. แสดงข้อมูลทั้งหมดจาก Sheet ในตาราง HTML
  4. ค้นหาข้อมูลด้วย keyword
  5. แก้ไขและอัปเดตข้อมูลที่มีอยู่ใน Sheet
  6. ลบข้อมูลพร้อม confirm dialog
Module 5: ระบบจริง Lead Form / Booking / PO Request 4 บท
  1. สร้างระบบ Lead Form / ลงทะเบียนครบวงจร
  2. สร้างระบบ Booking คิวหรือบริการ
  3. สร้างระบบ PO Request / ขออนุมัติ
  4. ปรับ template ให้เข้ากับงานของตัวเอง
Module 6: ส่งอีเมลอัตโนมัติ 4 บท
  1. ส่งอีเมลด้วย GmailApp.sendEmail()
  2. ออกแบบ email template แบบ HTML
  3. ส่งอีเมลแจ้งเตือนเมื่อมีข้อมูลใหม่
  4. ตั้ง Trigger อัตโนมัติเมื่อมีการบันทึกข้อมูล
Module 7: Deploy เป็น Web App 4 บท
  1. Deploy Apps Script เป็น Web App
  2. ตั้งค่าสิทธิ์ผู้ใช้: Anyone, Organization, Specific
  3. แชร์ link ให้คนอื่นใช้งาน
  4. อัปเดต Web App หลัง deploy แล้ว
Module 8: ใช้ AI ช่วยเขียนและแก้โค้ด 4 บท
  1. prompt ให้ AI เขียน Apps Script ได้ถูกต้อง
  2. แก้ error ด้วย AI เมื่อโค้ดพัง
  3. ใช้ Gemini ใน Google Sheets ช่วยสร้าง formula และ script
  4. แนวทางต่อยอด: ขยาย feature และดูแลระบบต่อเนื่อง
